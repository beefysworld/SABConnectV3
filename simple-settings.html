SABconnectV3 Settings

The URL where your SABnzbd server is running
Found in SABnzbd Config → General → API Key

Instructions

  1. Install SABnzbd if you haven't already
  2. Start SABnzbd - it usually runs on http://localhost:8080/
  3. Find your API Key in SABnzbd: Config → General → API Key
  4. Test the connection using the button above
  5. Save your settings once the test passes

Troubleshooting

  • Connection failed: Make sure SABnzbd is running
  • Wrong URL: Check if SABnzbd is on a different port
  • API errors: Try without API key first, then add it if needed
